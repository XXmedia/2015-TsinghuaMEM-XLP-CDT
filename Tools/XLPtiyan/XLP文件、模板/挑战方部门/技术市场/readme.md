基本规章制度
--------------------------------
技术市场交易流程与规范

1、商品要求
	1.1、卖家组成
		卖家为挑战方，为任务方提供技术培训、技术指导服务
	1.2、技术商品组成
		技术商品包括但不限于图像处理培训、PPT制作与演讲培训、git培训、软硬件问题指导等
	1.3、商品定价
		    培训按照时间计算。具体价格陈述在本目录的《XLP技术培训协议书》当中；
		    问题指导按问题个数计算，每一个问题视难度不同，收费不同，具体价格陈述在本目
		录的《XLP问题解决协议书》中
	1.4、技术培训时间安排
		1.4.1、免费基础课安排：按照日程表确定
		1.4.2、付费加强课安排：到技术市场签订《XLP技术培训协议书》时依照双方的自由时间商定
	1.5、技术指导时间安排
		签订协议后可以马上开始问题的解决过程，具体见相关协议书



2、交易要求
	2.1、交易记录
		买家和卖家自己记录每天的收入与支出并在需要的时候向银行报账，卖家将按日结算总账
	2.2、市场控制
		如果卖家判断买家有任何危害市场的行为，有权立即停止对买家的技术支持
		任务方自己交易技术登记并缴纳的费用为交易额的30%
		如果任务方不同组别之间存在未在市场登记的私下交易，一旦被查处将处以罚金3000元每次


3、权利维护
	买家如果发现卖家有欺诈行为，可以向监管部门提交相关诉讼

附注：文中如无特别说明，价格单位均为XLP规定标准单位XD（每个团队的起始货币总额为60000XD）
